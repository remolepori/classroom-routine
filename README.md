# Classroom Routine (Wordpress Plugin)

## Beschreibung
Visuelles Tages-Board mit Icons, Texten, pro-Benutzer-Speicherung, Plus-Button und Speichern/Laden-Menü.

## Shortcode
<code>[classroom_routine]</code>
